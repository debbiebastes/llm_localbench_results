A company is currently using an On-Demand EC2 instance for their application which they plan to migrate to a Reserved EC2 Instance to save on cost. Which of the following is the most cost-effective option if the application being hosted would be used for more than 3 years?	

(A) All Upfront Convertible Reserved Instance pricing for a 1-year term.
(B) No Upfront Convertible Reserved Instance pricing for a 3-year term.
(C) All Upfront, Standard Reserved Instance pricing for a 3-year term.
(D) No Upfront Standard Reserved Instance pricing for a 1-year term that is renewed every year.

Answer: 
(B) No Upfront Convertible Reserved Instance pricing for a 3-year term.

Rationale: The No Upfront, Convertible Reserved Instance pricing option provides the most cost savings if the application being hosted would be used for more than three years. The upfront payment is waived, and there is an hourly charge based on usage, which can result in significant cost savings over time.

(C) No Upfront, Convertible Reserved Instance pricing for a 1-year term that is renewed every year.
(D) All Upfront Standard Reserved Instance pricing for a 3-year term.

Answer: (B) No Upfront, Standard Reserved Instance pricing for a 3-year term.

Rationale: The No Upfront, Standard Reserved Instance pricing option provides the most cost savings if the application being hosted would be used for more than three years. There is no upfront payment required and the hourly charge based on usage can result in significant cost savings over time.
