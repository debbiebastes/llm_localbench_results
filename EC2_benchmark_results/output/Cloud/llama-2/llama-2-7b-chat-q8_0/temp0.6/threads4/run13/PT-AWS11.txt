Which AWS service enables you to build event-driven applications and decouple the components of your application architecture?

(A) Amazon SQS
(B) Amazon EventBridge (Amazon CloudWatch Events)
(C) Amazon SNS
(D) Amazon Kinesis

Answer: 
The AWS service that enables you to build event-driven applications and decouple the components of your application architecture is (B) Amazon EventBridge (Amazon CloudWatch Events).
